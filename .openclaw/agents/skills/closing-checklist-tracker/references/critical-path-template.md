# Critical Path Template

## Purpose

Map the minimum-time sequence of dependent tasks from PSA execution to closing. Any delay on the critical path delays closing. Non-critical tasks have float -- they can slip without affecting the close date.

## 30-Day Closing Critical Path

Used for: all-cash acquisitions, 1031 exchanges, or transactions with pre-arranged financing.

-29  | Order PCA (rush)                          | Buyer          | [ ]    | Rush = 2 week turnaround
-29  | Request seller document package           | Buyer          | [ ]    |
-28  | Receive seller docs (rent roll, T12, leases)| Seller       | [ ]    | 48-hr delivery required
-25  | Property tour and unit inspections         | Buyer          | [ ]    |

**30-Day Critical Path Chain:**
PSA --> Title + Phase I + PCA (parallel, all rush) --> DD Review --> DD Resolution --> Closing Docs --> Wire --> Close

**Float items (non-critical, can be completed post-close if needed):**
- Entity formation (can be done in parallel, 3-5 days)
- Insurance binding (5-7 days lead time, non-critical until day -7)
- Property management engagement (can finalize post-close)
- Service contract review (can assume all and renegotiate post-close)

## Extension Protocol

When closing will not occur on the scheduled date:

1. **Identify the cause**: Financing delay? Title issue? Seller cure? Third-party report?
2. **Assess the timeline**: How many additional days are needed? Is the cause fixable or is there a structural problem?
3. **Check PSA extension provisions**: Most PSAs allow 1-2 extensions of 15-30 days, sometimes with additional earnest money deposit.
4. **Notify all parties immediately**: Lender (rate lock extension cost), seller (extension request), title (revised closing date), property manager (delay transition).
5. **Rate lock extension cost**: Typically 0.125-0.250% per 15-day extension. Budget for this.
6. **Document the extension**: Written amendment to PSA signed by both parties.

Common causes of delay (and prevention):
- Appraisal below purchase price: Engage lender early, provide strong comp package
- Lender underwriting delay: Submit complete package upfront, follow up weekly
- Title defect requiring cure: Order title on day 1, flag issues immediately
- Estoppel non-delivery: Start requesting on day -45, not day -25
- Environmental issue: Order Phase I on day 1, always
