# Contract Negotiation Guide

## Purpose

Reference for negotiating GMP, lump sum, and cost-plus construction contracts. Covers standard AIA modifications, fee structures, retainage, insurance, bonding, and dispute resolution for institutional CRE development.

## Contract Structure Selection

### When to Use Each Type

| Structure | Best For | Risk Profile | Design Completeness |
|-----------|----------|-------------|-------------------|
| **GMP** | Most institutional projects, ground-up, major renovation | Shared risk -- GC bears overrun, owner benefits from savings | 60-90% CD (can start with partial drawings) |
| **Lump Sum (Stipulated Sum)** | Tenant fit-out, simple renovation, well-defined scope | Risk transferred to GC (priced in) | 100% CD required |
| **Cost-Plus Fixed Fee** | Fast-track, phased, or highly uncertain scope | Owner bears cost risk, GC has no overrun incentive | Can start at 30% DD |
| **Cost-Plus with GMP** | Hybrid -- starts cost-plus, converts to GMP at buyout | Transitional risk sharing | Starts at 50% DD, GMP set at 90% CD |

### Fee Structure Benchmarks

| Component | Market Range | Institutional Target | Notes |
|-----------|-------------|---------------------|-------|
| GC Fee (profit) | 3.0-5.0% | 3.5-4.0% | Of cost of work, excluding contingency |
| GC Overhead | 8.0-12.0% | Included in general conditions | Some GCs break out, others bundle |
| General Conditions | 5.0-8.0% | 5.5-7.0% of hard cost | Staffing, temp facilities, equipment |
| GC Contingency | 2.0-5.0% | 2.5-3.0% | For GC's internal risk; burns down during buyout |
| Owner Contingency | 3.0-7.0% | 3.0-5.0% | Not in GC's contract; owner holds separately |
| Preconstruction Fee | $50K-$250K | Based on scope/duration | Credited against GC fee if awarded construction |

## GMP Provisions

### Shared Savings

The GMP shared savings clause is the most negotiated provision. Standard structures:

| Split | Owner : GC | When Used |
|-------|-----------|-----------|
| 50/50 | Equal split | Most common starting point |
| 60/40 | Owner-favorable | Institutional owner with leverage |
| 75/25 | Owner-heavy | Large projects, competitive market |
| 100/0 | All to owner | Rare; GC has no savings incentive |

**Key provision language**: "Savings shall be calculated as the difference between the final GMP (as amended) and the actual Cost of the Work plus the Contractor's Fee. Savings shall be distributed X% to Owner and Y% to Contractor within 60 days of final accounting."

**Negotiate**: Savings calculation should exclude owner contingency draws and should be calculated after all GC contingency is reconciled.

### GMP Amendment Triggers

Define when the GMP can be increased. Standard triggers:

1. **Owner-directed changes**: Changes to scope initiated by owner (always a trigger)
2. **Unforeseen conditions**: Concealed or unknown conditions materially different from contract documents (requires documentation)
3. **Force majeure**: Events beyond contractor's control (weather beyond historical norms, labor strikes, material unavailability)
4. **Regulatory changes**: Code changes enacted after bid date that affect scope
5. **Design errors and omissions**: Errors in construction documents that require additional work (owner typically bears this risk)

**Negotiate**: Require GMP amendments to be approved in writing before work proceeds. Set a threshold (e.g., $25K or 0.5% of GMP) below which the PM can approve, above which requires owner principal approval.

### Buyout Savings Disposition

When the GC buys out subcontracts below the GMP line items:

| Approach | Description | Owner Impact |
|----------|-------------|-------------|
| **Flows to savings** | Buyout savings accumulate and are shared per savings split | Best for owner -- creates shared incentive pool |
| **Reduces GMP** | Buyout savings reduce GMP dollar-for-dollar | Owner captures 100% but removes GC incentive |
| **GC retains** | GC keeps buyout savings | Worst for owner -- avoid |
| **Hybrid** | First $X flows to GMP reduction, remainder shared | Compromise approach |

**Best practice**: Require open-book buyout process. Owner has right to attend sub buyout meetings. All sub bids visible. Buyout savings flow to shared savings pool.

### Allowance Reconciliation

Allowances in the GMP for items not yet fully designed or priced:

- Track each allowance separately with running balance
- Overruns require GMP amendment; underruns flow to savings
- Owner must approve expenditure against allowance before work proceeds
- Common allowances: FF&E, signage, IT/AV, landscaping, utility connections, abatement

## Cost-Plus Fee Structures

### Fixed Fee

GC receives a fixed dollar amount regardless of final cost. Incentivizes cost control only if combined with GMP conversion.

- Fee typically calculated as percentage of estimated cost of work at contract execution
- Fee does not increase if costs increase (unless scope changes)
- Fee is earned proportionally to percentage of work complete

### Percentage Fee

GC receives a percentage of actual cost of work. Creates perverse incentive -- higher cost = higher fee.

- Use only for emergency work or when scope is truly unknowable
- Cap the fee at a dollar amount to limit exposure
- Typical range: 5-8% for small projects, 3-5% for large projects

### Fee on Change Orders

| Item | Subcontractor | GC on Sub Work | GC on Self-Performed |
|------|--------------|----------------|---------------------|
| Overhead | 10% | -- | 10% |
| Profit | 5% | 5% | 5% |
| **Total Markup** | **15%** | **5%** | **15%** |

**Critical**: No markup on markup. If a sub marks up 15%, the GC's 5% applies to the sub's base cost, not the marked-up amount.

| Risk | Owner | GC | Shared |
|------|-------|----|--------|
| Design completeness | X | | |
| Means and methods | | X | |
| Concealed conditions | X | | |
| Code compliance of design | X | | |
| Code compliance of construction | | X | |
| Schedule (weather) | | | X |
| Material price escalation | | X | |
| Labor availability | | X | |
| Subcontractor default | | X | |
| Permit delays | | | X |
| Utility connection delays | | | X |

### Key Modifications for Institutional Owners

1. **Concealed conditions**: Standard AIA places risk on owner. Consider sharing: first $50K on GC, remainder on owner.
2. **Allowances**: Even in lump sum, include allowances for unknowable items. Reconcile at completion.
3. **Unit prices**: Establish unit prices for likely change order items (e.g., rock excavation per CY, additional piles per LF).
4. **Escalation**: For projects with long preconstruction, include an escalation provision pegged to an index (ENR or BLS CPI for construction).

## Retainage Negotiation

### Standard Retainage Structure

| Phase | Retainage % | Notes |
|-------|------------|-------|
| Start through 50% complete | 10% | Standard institutional |
| 50% complete to substantial | 5% | Reduction incentivizes performance |
| Substantial completion | Release to 150-200% of punch list value | Retain enough to ensure closeout |
| Final completion | Release balance | After all closeout docs received |

### Negotiation Points

- **GC request**: 5% flat from start. **Counter**: 10% to 50%, then 5%. Compromise: 7.5% flat.
- **Sub retainage**: GC should pass through same retainage terms to subs. Verify in subcontracts.
- **Retainage on stored materials**: Typically no retainage on materials stored on-site (already paid for and insured).
- **Interest on retainage**: Some jurisdictions require interest. Check state law.
- **Retainage escrow**: For large projects, owner may escrow retainage in interest-bearing account.

## Insurance Requirements

### Owner's Minimum Insurance Requirements for GC

| Coverage | Minimum Limit | Notes |
|----------|--------------|-------|
| Commercial General Liability | $1M per occurrence / $2M aggregate | Must name owner as additional insured |
| Automobile Liability | $1M combined single limit | Owned, hired, non-owned |
| Workers' Compensation | Statutory limits | Per state law |
| Employer's Liability | $1M per accident | |
| Umbrella/Excess | $5-25M | Based on project size; $25M for >$50M projects |
| Professional Liability | $2M (if design-build) | Only if GC has design responsibility |
| Pollution Liability | $2M (if abatement) | Required if environmental work in scope |

### Builder's Risk

- Owner typically procures builder's risk policy for the project
- Coverage: all-risk, replacement cost, including materials in transit and stored off-site
- Named insureds: owner, GC, all subcontractors
- Deductible: $10K-$50K standard; negotiate who bears deductible
- Duration: from notice to proceed through substantial completion + 60 days
- Soft costs coverage: include architect fees, financing costs, lost rental income for delay

### OCIP vs CCIP

| Feature | OCIP (Owner-Controlled) | CCIP (Contractor-Controlled) |
|---------|------------------------|----------------------------|
| Who controls | Owner | GC |
| Premium in GC bid | No -- stripped out | Yes -- included |
| Cost savings | 1-3% of hard cost typical | Savings retained by GC |
| Administrative burden | On owner | On GC |
| Best for | Large projects (>$50M) | Small-medium projects |
| Claims management | Owner controls | GC controls |

## Bonding Requirements

### Standard Bond Requirements

| Bond | Amount | Purpose |
|------|--------|---------|
| Performance Bond | 100% of contract value | Guarantees GC completes work |
| Payment Bond | 100% of contract value | Guarantees GC pays subs and suppliers |
| Bid Bond | 5-10% of bid amount | Guarantees GC honors bid if selected |
| Maintenance Bond | 1-2 years | Guarantees warranty work (sometimes required) |

### Surety Evaluation

- A.M. Best rating: A- or better (A preferred)
- Listed on U.S. Treasury Department Circular 570
- Single project limit > contract value
- Aggregate limit sufficient for GC's total portfolio
- Verify directly with surety -- do not rely on GC's representation

### Subcontractor Bonds

Require payment and performance bonds from subcontractors on critical trades:
- Mechanical (HVAC): if > $2M
- Electrical: if > $1.5M
- Plumbing: if > $1M
- Elevator: always (proprietary systems, long lead)
- Curtain wall/building envelope: if > $2M
- Fire protection: if > $500K

## Dispute Resolution

### Escalation Ladder

1. **Project-level resolution**: PM and superintendent attempt resolution within 7 days
2. **Executive-level resolution**: Owner principal and GC principal meet within 14 days
3. **Mediation**: Non-binding mediation with agreed mediator within 30 days
4. **Binding arbitration or litigation**: Per contract election

### Mediation vs Arbitration vs Litigation

| Method | Timeline | Cost | Privacy | Appeal Rights |
|--------|----------|------|---------|--------------|
| Mediation | 30-60 days | Low ($5-15K) | Private | N/A (non-binding) |
| Arbitration (AAA) | 6-12 months | Medium ($50-200K) | Private | Very limited |
| Litigation | 18-36 months | High ($200K+) | Public | Full appeal rights |

**Institutional preference**: Mediation first (mandatory), then binding arbitration (AAA Construction Rules). Avoids public litigation and is faster. Include carve-out for mechanic's lien claims (these go to court regardless).

### Continuation of Work

**Critical provision**: Contractor must continue performing during dispute resolution. Payment of undisputed amounts continues. Disputed amounts held in escrow pending resolution. This prevents the GC from using work stoppage as leverage.
